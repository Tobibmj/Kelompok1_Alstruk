
Isi dari project ini meliputi beberapa bagian  dalam pemrograman Python, yaitu:


1. List
Contoh membuat list, mengakses elemen, menambah item, menghapus item, dan operasi dasar lainnya.

2. Dictionary
Cara membuat dictionary, mengambil nilai berdasarkan key, menambah data baru, mengupdate, dan menghapus key.

3. Set
Contoh penggunaan set, menambah item, menghapus item, serta operasi penting seperti union & intersection.

4. Tuple
Cara membuat tuple, mengakses elemen, dan karakteristik tuple yang bersifat immutable.

5. Function (def)
Penjelasan cara membuat fungsi, memberikan parameter, mengembalikan nilai, dan contoh pemanggilan fungsi.
